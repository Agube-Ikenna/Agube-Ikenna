## About Me

I am an MSc Environmental Toxicology student working with field-study data, leaf decomposition analysis, water-quality variables, and reproducible research workflows. I also build web tools that make everyday data easier to understand and act on.

- Research focus: environmental toxicology, freshwater ecology, field data, and statistical analysis
- Technical focus: R, React, JavaScript, data visualization, reproducible project structure
- Current direction: combining scientific analysis with clean software presentation for research and applied roles
